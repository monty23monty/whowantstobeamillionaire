# Who Wants to Be a Millionaire (Flask)

A web-based Python implementation of **Who Wants to Be a Millionaire** with:
- User registration and login
- Custom question set creation
- Exactly 15 multiple-choice questions per set
- Game creation from a selected question set
- Full game flow with winnings/checkpoint logic

## Quick Start

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python run.py
```

Visit `http://localhost:5000`.

## Testing

```bash
pytest
```
